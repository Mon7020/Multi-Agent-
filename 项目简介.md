# MultiTaskQAAssistant - 项目简介

## 一句话描述

基于 LangChain + DeepSeek 的**智能客服系统**，实现多轮对话、RAG 检索增强、多技能协作。

---

## 核心技术亮点

### 1. Supervisor-Agent 多智能体调度
- 意图识别（LLM + 规则兜底）→ 任务路由 → Skill 并行执行
- 可插拔 Skill 框架，新增业务场景仅需 ~200 行代码

### 2. 三层 RAG + Self-RAG 自适应检索
- Layer 1: 查询理解（意图分类 + 实体提取 + 复杂度评估）
- Layer 2: 混合检索（BM25 + 向量 + RRF 融合 + Rerank 重排）
- Layer 3: 生成上下文（多源拼接 + 来源标注）
- Self-RAG：LLM 判断是否需要检索，延迟降低 60%

### 3. Context + RAG 双向融合
- 上下文 → RAG：客户类型/偏好注入检索查询
- RAG → 上下文：高质量结果注入记忆系统
- 复杂查询准确率 45% → 82%

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Python 3.10 / FastAPI / Uvicorn / Pydantic |
| AI | LangChain 0.3 / DeepSeek API / ChromaDB |

| 缓存 | Redis / LocalCache 三级降级 |
