# 用户操作手册

## 适用范围

这份手册面向普通前台用户，说明登录后可以使用哪些能力，以及哪些配置已经迁移到后台。

## 你可以做什么

登录用户前台后，可以看到三个标签页：

- 对话
- 知识库
- 设置

## 知识库页面

当前知识库页面是只读模式。

你只能看到满足以下条件的文件：

- 已发布
- 后台允许前台显示
- 你的角色在允许访问范围内

你可以做的操作：

- 查看文件列表
- 打开文档内容
- 阅读文本内容

你不能做的操作：

- 上传文件
- 编辑文件
- 删除文件
- 发布文件
- 修改可见范围

如果某个文件看不到，通常是以下原因之一：

- 文件仍处于草稿状态
- 后台已将其设置为隐藏
- 你的角色不在允许访问范围内

## 设置页面

前台设置页现在是只读摘要。

你可以看到：

- 当前账号角色和状态
- 系统运行参数摘要
- 后端健康状态和版本
- 前台 / 后台的能力边界说明

你不能在前台修改：

- RAG 运行参数
- 知识库发布策略
- 权限模型
- 缓存与检索策略

这些变更必须由后台管理员完成。

## 登录与权限

角色说明：

- `user`：普通前台用户
- `operator`：运营
- `admin`：管理员
- `super_admin`：超级管理员

说明：

- 普通 `user` 无法进入后台
- 即使是高角色用户，在前台看到的知识库也仍然是只读视图

## 常见问题

### 为什么我看不到某个知识文件？

请联系管理员确认：

- 文件是否已发布
- 文件是否被设置为前台可见
- 当前角色是否在允许访问列表中

### 为什么设置页不能修改参数？

这是当前系统设计的一部分。运行参数与权限策略统一收口到后台，避免前台误操作影响全局系统。
